# ngslides
Use Angular to create presentation.

# Usage
Use  <lib-ngslides> to create a slide inside your <app-ngpres> presentation:

```
<app-ngpres>
  <lib-ngslides>
    <div class="center">
      AngularAI
    </div>
  </lib-ngslides>

  <lib-ngslides>
    Slide 2
  </lib-ngslides>

  <lib-ngslides>
    Slide 3
  </lib-ngslides>

  <lib-ngslides>
    Slide 4
  </lib-ngslides>
</app-ngpres>
```
#Tools
 1. Press 'w' to move one slide up
 2. Press 's' to move one slide down
